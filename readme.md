## üåü Key Features
* **W-2 Support:** Quickly input wages and federal withholding.
* **1099 Integration:** Dedicated sections for:
    * **1099-INT:** Interest income from banks/CDs.
    * **1099-DIV:** Dividend income (essential for assets like **JEPI**).
    * **1099-B:** Capital gains from stock and brokerage sales.
    * **1099-DA:** Support for the **new 2026 Crypto reporting requirements**.

* **Privacy First:** Includes a "Clear All Data" function to wipe the session state immediately.
